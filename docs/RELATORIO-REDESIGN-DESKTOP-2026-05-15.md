# Redesign desktop 2026-05-15

## Objetivo
Reformular a aparência do site no computador para deixar a navegação com comportamento de site tradicional, mantendo a identidade visual escura com destaque dourado/oliva do Universo Segurança Pública.

## Alterações aplicadas
- Criada a camada `css/site-desktop-redesign.css` com escopo principal para telas a partir de 900px.
- Adicionada navegação horizontal de desktop no cabeçalho.
- Ocultadas, no desktop, a navegação inferior e os botões flutuantes de estilo aplicativo.
- Reestilizado o cabeçalho para funcionar como hero institucional amplo.
- Reorganizados cards e seções da página inicial em grade desktop.
- Reestilizado o menu aberto no computador como painel superior, mantendo a vitrine compacta de produtos.
- Mantida a identidade visual atual sem uso da cor roxa da referência.

## Observações
- As mudanças anteriores foram preservadas: buscador de carreira, remoção de Base Legal/Poderes e Deveres, remoção dos chamativos Anuncie Aqui, menu superior e correções da home/concursos.
- A responsividade mobile foi preservada, mas o foco desta etapa foi o computador.
