# Visual App Suave — UniSegPub

Esta versão aplica uma camada visual inspirada em interface mobile clara/neumórfica, semelhante à referência enviada.

## O que foi alterado

- Adicionado `css/app-soft-mobile.css`.
- Adicionados controles flutuantes superiores: voltar/início e abrir menu.
- Adicionada navegação inferior fixa estilo aplicativo.
- Header, cards, botões, tabelas, sidebar e rodapé receberam aparência de cartões suaves com cantos arredondados e sombra.
- A lógica JavaScript principal foi preservada. A camada é visual, para reduzir risco de quebrar funcionalidades.

## Arquivos principais

- `css/app-soft-mobile.css` — nova aparência.
- Todos os `.html` receberam o link para esse CSS e a navegação inferior.

## Observação

Esta versão é uma proposta visual. Caso queira aplicar em produção, teste as páginas principais e ajuste anúncios/cores conforme conversão.
