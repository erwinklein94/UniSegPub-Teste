# Universo Segurança Pública — versão profissional

Estrutura final do site:

- `index.html`
- `assets/css/style.css`
- `assets/js/dados.js` — camada de dados
- `assets/js/app.js` — navegação, header, tabelas, direitos, concursos, comparador e contato
- `assets/img/` — imagens locais usadas pelo HTML

Melhorias aplicadas nesta versão:

1. Visual institucional mais sério, com paleta sóbria em azul-marinho, grafite e dourado.
2. Página inicial redesenhada para orientar melhor o usuário logo na entrada.
3. Cards, botões, formulários, tabelas e menu lateral com aparência mais consistente e profissional.
4. Ícones em estilo textual/institucional no lugar de emojis, deixando a interface mais madura.
5. Melhorias de acessibilidade: link de pular conteúdo, foco visível, `aria-current`, `aria-hidden` e navegação por teclado.
6. Melhor experiência em celular, com CTAs mais claros, melhor espaçamento e cards mais legíveis.
7. `assets/` organizado diretamente na raiz do projeto, sem depender do `assets.zip` interno.

Observação: o conteúdo continua informativo e deve ser conferido em fontes oficiais antes de decisões funcionais, jurídicas ou financeiras.
